# ゲーム設計メモ（暫定）

## 目的
- ロックマンX系の2Dアクションを作る
- エンジンに頼らず、処理を自分で書く
- DXライブラリ + C++

---

## 技術スタック
- 言語: C++
- 描画: DXライブラリ
- IDE: Visual Studio
- バージョン管理: Git + GitHub

---

## 画面・描画
- 表示: フルスクリーン（PC）
- 内部解像度: 320x180（予定）
- 描画方式: バックバッファ → 整数倍拡大
- フレームレート: 60fps 固定

---

## キャラクター
- ドット絵アニメーション
- 複数枚の静止画をループ表示
- 状態によってアニメーション切り替え

### 状態（予定）
- Idle（待機）
- Run（左右移動）
- Jump（空中）
- Fall（落下）
- Land（着地）

---

## 当たり判定
### キャラ
- 矩形（AABB）
- アニメフレームごとに当たり判定を切り替える

### 地形
- タイル単位で判定
- 全ピクセル判定はしない
- 一部のみ当たり判定を持つタイルあり

---

## 移動・物理
- 重力あり
- 段差は「小段差のみ自動で乗り越え」
- 大きな段差はジャンプ必須

---

## マップ
- CSV で管理
- タイルIDで地形を定義
- 当たり判定はタイル側に持たせる

---

## 実装順（予定）
1. ゲームループ
2. 入力管理
3. プレイヤー移動（左右 + 重力）
4. 接地判定
5. 状態遷移
6. アニメーション
7. マップ読み込み

---
